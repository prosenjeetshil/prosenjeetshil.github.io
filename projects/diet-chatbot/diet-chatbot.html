    • Personal Diet Advisor Chatbot

    Description:

    The Personal Diet Advisor Chatbot is an AI-powered chatbot developed using Chatbot Kit, Next.js, and the OpenAI API. It assists users by providing intelligent diet-related advice and suggestions based on user input and queries.


    Technologies Used:

    • Next.js
    • Chatbot Kit
    • OpenAI API (GPT-3)
    • JavaScript
    • CSS

    Project Learnings:

    Working with the OpenAI API (GPT-3), chatbot interaction design, asynchronous handling in Next.js, Chatbot Kit integration, and enhancing UX through conversational AI.

    Project Challenges:

    Prompt engineering, API token handling, ensuring accurate dietary advice, response latency, and maintaining natural conversation flow.
